# 硬件性能测评指南  

本测试用于评估智能芯片的**访存性能**和**通信性能**。每个测试程序均位于独立文件夹中，按照以下步骤执行：  

1. **进入测试程序目录**  
2. **运行 `build.sh` 编译**（生成可执行文件在 `build/` 目录下）  
3. **执行测试程序**  

---

## **1. 访存性能测试**  
测试计算单元与芯片内存（如DRAM）之间的数据传输带宽（GB/s）。  

### **测试程序说明**  
| 测试程序 | 功能描述 | 执行方式 |  
|----------|---------|----------|  
| **`cuda-memcpy/`** | 测试内存拷贝（Host↔Device）带宽 | `./build/cuda-memcpy` |  
| **`gpu-cache/`** | 测试GPU缓存（L1 Cache）访问性能 | `./build/gpu-cache` |  
| **`gpu-l2-cache/`** | 测试GPU二级缓存（L2 Cache）访问性能 | `./build/gpu-l2-cache` |  
| **`cuda-stream/`** | 使用Copy/Scale/Add/Triad四种模式测试GPU显存读写带宽 | `./build/stream` |  

---

## **2. 通信性能测试**  
测试设备间或设备内部的通信带宽。  

### **测试程序说明**  
| 测试程序 | 功能描述 | 执行方式 |  
|----------|---------|----------|  
| **`bandwidthTest/`** | 测试PCIe/NVLink等通信带宽 | `./build/bandwidthTest` |  

---

## **示例测试流程**  
```bash
# 1. 进入测试目录（如访存测试）
cd cuda-memcpy/

# 2. 编译测试程序
./build.sh

# 3. 运行测试
./build/cuda-memcpy
```

请根据实际需求选择合适的测试程序，并记录测试结果用于性能分析。
